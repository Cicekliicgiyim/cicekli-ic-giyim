# 🌸 Atilla Çiçekli İç Giyim

**Atilla Çiçekli İç Giyim**, HTML, CSS ve JavaScript kullanılarak geliştirilmiş basit ve işlevsel bir e-ticaret web sitesidir. Kullanıcılar ürünleri görüntüleyebilir, sepete ekleyebilir, sipariş verebilir ve sipariş geçmişlerini inceleyebilir. Ayrıca admin kullanıcılar ürün yönetimi yapabilir.

---

## 🚀 Özellikler

- 👗 Ürün listeleme ve filtreleme
- 🛒 Sepet sistemi ve toplam fiyat hesaplama
- ✅ Sipariş onay ve geçmiş sayfası
- 🔐 Kullanıcı kayıt / giriş sistemi
- 🧑‍💼 Admin paneli (ürün ekleme, düzenleme, silme)
- 🔓 Oturum kontrolü ve kullanıcıya özel görünüm
- 💾 LocalStorage üzerinde veri yönetimi (sunucu gerekmez)

---

## 📂 Dosya Yapısı
```
cicekli-ic-giyim/
├── index.html          # Anasayfa
├── products.html       # Ürünler sayfası
├── cart.html           # Sepet sayfası
├── checkout.html       # Sipariş tamamlama
├── success.html        # Sipariş başarılı
├── login.html          # Giriş sayfası
├── register.html       # Kayıt sayfası
├── admin.html          # Admin paneli
├── style.css           # Ana stil dosyası
├── form.css            # Form stilleri
└── js/
    ├── main.js         # Genel işlevler
    ├── products.js     # Ürün listeleme
    ├── cart.js         # Sepet işlemleri
    ├── checkout.js     # Sipariş işlemleri
    ├── success.js      # Sipariş onay
    ├── login.js        # Giriş işlemleri
    ├── register.js     # Kayıt işlemleri
    └── admin.js        # Admin işlemleri
```

## 🛠️ Kurulum

1. Repoyu klonlayın:
```bash
   git clone https://github.com/kullanici-adi/cicekli-ic-giyim.git
```

2. Proje klasörüne girin:
```bash
   cd cicekli-ic-giyim
```

3. `index.html` dosyasını bir tarayıcıda açın veya [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) uzantısını kullanın.

> ⚠️ Sunucu gerekmez. Tüm veriler `localStorage` üzerinde tutulur.
